## Special Features

### Safe Word
- Any player can pause the game at any time
- All players can take a break
- Resume when ready

### Consent Tools
- Skip button on every action
- No judgment for skipping
- All actions are consensual

### Privacy
- No data stored
- No logging of actions
- Anonymous play
- Private rooms only
